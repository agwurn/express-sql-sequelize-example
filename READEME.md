## To Start

### install dependencies
```
npm install
```
### start the server
```
nodemon app.js
```

### If you haven't install MySQL before
```
brew install mysql
brew services start mysql

<!-- set the password to "password" -->
mysql_secure_installation

<!-- enter the mysql cli -->
mysql -u root -p
CREATE DATABASE my_express_sequelize_test;

Exit;
```
